Interactividad en Aplicaciones Web Modernas

Las páginas y aplicaciones web modernas se enfoccan en gran medida a la interacción con sus usuarios. El auge de lenguajes de programación enfocados en el cliente como JavaScript permiten interactuar con el usuario de manera eficaz, responsiva y creativa. Las aplicaciones web no solo se limitan a tomar datos de entrada como nombres o números, también toman la posición, movimiento y comportamiento del puntero. Muchas aplicaciones ahora pueden tomar datos de otras para facilitar la navegación. Incluso existen algunas que emulan aplicaciones de escritorio como Photoshop, editores de vídeo o incluso sistemas operativos.
